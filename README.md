# EAIDA

Bu repo, IDA/IHA tarafinda yer istasyonu, telemetri ve otonom denemeler icin gelistirilen kodlari icerir.

## Dosya ve Klasor Amaçlari

- emir_parkur/
	- m2.py: ROS2 + MAVLink tabanli otonom parkur/navigasyon denemesi.
	- max_py: Alternatif veya eski deneme dosyasi.

- yer_istasyonu/
	- emirxd.py: PyQt5 tabanli yer kontrol istasyonu (telemetri, harita, temel kontrol).

- gereksiz/
	- eaida.py: Eski/alternatif yer istasyonu surumu.
	- iha+ida.py: IHA + IDA birlikte calisan eski/alternatif surum.
	- elif+: Arayuz ve telemetri icin alternatif deneme kodu.
	- elifida: Arayuz ve telemetri icin alternatif deneme kodu.

## Not

Bu repoda aktif gelisim ve deneme dosyalari birlikte bulunur.
